<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added AI-powered conversation branching for deeper Ramayana explorations, allowing you to ask follow-up questions about characters like Rama or Sita.
- Introduced customizable digital avatars with voice modulation, supporting multiple regional accents for Hanuman and other figures.
- New reels generator creates short, shareable videos from any episode summary.

## Improvements

- Enhanced episode rendering speed by `>40%` using optimized AI models.
- Improved mobile responsiveness for interactive stories.

## Bug Fixes

- Fixed avatar lip-sync issues during long dialogues.
- Resolved intermittent failures in story generation API calls.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Launched interactive quizzes on Ramayana values, integrated with your conversation history.
- Added export functionality for conversations as PDF or audio files.

## Bug Fixes

- Patched security vulnerability in avatar upload endpoints.
- Corrected timeline inaccuracies in episode visualizations.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## Initial Release

- Core platform with AI conversations, digital avatars, and episode generation.
- Basic reels and stories creation tools.
- User authentication via `https://auth.ramayanam.ai`.

## Breaking Changes

- Updated API base URL to `https://api.ramayanam.ai/v1`.

</Update>

<Expandable title="How to Migrate Between Versions" default-open="false">

If upgrading from v1.0.0, update your API client:

````javascript
// Old endpoint
const response = await fetch('https://api.ramayanam.ai/episodes');

// New endpoint (v1.2.0)
const response = await fetch('https://api.ramayanam.ai/v1/episodes?avatar=hanuman');
````

</Expandable>
