## Overview

Ramayanam.ai reimagines the timeless Indian epic, Ramayana, through artificial intelligence. You engage with digital avatars of iconic characters like Lord Rama and Sita, explore interactive episodes, reels, and stories, and participate in AI-powered conversations that reveal profound values such as dharma, devotion, and righteousness. This platform bridges ancient wisdom with modern technology, making the epic accessible and immersive for everyone.

## Embed Ramayanam.ai Conversations

Integrate AI chats into your website or app.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.ramayanam.ai/v1/chat', {
    method: 'POST',
    headers: { 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' },
    body: JSON.stringify({ message: 'Tell me about Hanuman', character: 'hanuman' })
  });
  const data = await response.json();
  console.log(data.reply);
  ```
  ```python
  import requests

  response = requests.post(
      'https://api.ramayanam.ai/v1/chat',
      headers={'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json'},
      json={'message': 'Tell me about Hanuman', 'character': 'hanuman'}
  )
  print(response.json()['reply'])
  ```
</CodeGroup>
